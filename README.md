# DevConsole

AI-driven orchestration platform for automatic development and testing of:

- websites;
- backend services;
- Linux servers;
- Android/Flutter applications.

## Stack

- FastAPI
- OpenAI Responses API
- SQLite/PostgreSQL
- Docker
- Android SDK
- Flutter SDK
- Git integration
- WebSocket live logs

## Planned modules

- AI Agent Orchestrator
- Workspace Manager
- Android Builder
- Git Automation
- Release Assistant
- Live Console
- Snapshot/Rollback Engine
- Persistent Dialog Database
- Task Queue
- Device Launcher
- SSH Runner

## Architecture

```text
frontend/
backend/
workers/
storage/
projects/
logs/
```
